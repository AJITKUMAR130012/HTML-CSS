Block Element

It start the word from the new line and it also take the all the left side space. It any word start after this is start from the new line.

Example of the Block element:

This is

my
car

Inline Element

It start the word from the continueation of the same line and it doesn't take the all the left side space. It take the space that is required for that word.

This is example of the Inline:

This is my car
