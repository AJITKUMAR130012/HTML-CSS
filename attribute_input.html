Various atribute of the input tag

1.Placeholder: It will give hint to the user of input format.
2.Value: Value is same sa format but it is editable.
3.Disabled: It will disable the form for that input.
4.Size: It will increase the form length or could be decrease also.
5.Maxlength: It will fix the maximum length of the character.
6.Min/Max: It will give maximum and minimum limit to the number.
7.Multiple: It can select multiple file.
8.Required: It will make that field essential for the form.
9.Step: It will make the form to make input multiple of the step value.
10.Autofocus: When page open first time then crusor point to that form.
